🏨 Bristol Hotel website testing

I tested the hotel website bristolkrasnodar.ru and prepared different types of testing documentation:


Test plan describing what to test and how to test (scope, types of testing, risks, pass/fail criteria, environment).
Checklist for quick verification of the site's main sections: Home Page, Rooms, Booking, Discounts, Contacts, Gallery, Navigation and Responsive design.
Test cases created via Test IT, covering key user flows: navigation, room browsing, booking (including negative scenarios), contacts, discounts, gallery and services.
Bug report example created via Jira, with steps to reproduce, screenshots, severity and priority.
